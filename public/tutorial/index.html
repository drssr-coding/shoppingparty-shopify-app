Getting Started with Shopping Party

1. Installation

  • Install Shopping Party from the Shopify App Store
  • Grant necessary permissions
  • Configure basic settings

2. Creating Your First Shopping Session

  1. Navigate to the Shopping Party dashboard
  2. Click "Create Shopping Session"
  3. Set session parameters (duration, max participants)
  4. Share the invitation link with participants

3. Managing Sessions

During a shopping session, you can:

  • Monitor active participants
  • Share products with the group
  • Use the chat feature
  • End the session manually

4. Best Practices

  • Keep sessions focused and time-limited
  • Encourage participant interaction
  • Use chat moderation features when needed
  • Follow up with participants after sessions
